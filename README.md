### 2. AI기반 챗봇 및 OCR 개발 전문가 과정

* **SQL** : 데이터를 수집하고 정제하는 데 중요한 역할, 분석 및 모델링에 필요한 데이터를 효과적으로 관리

* **Computer Vision** : 딥러닝 알고리즘을 사용하여 이미지 기반 데이터를 처리
